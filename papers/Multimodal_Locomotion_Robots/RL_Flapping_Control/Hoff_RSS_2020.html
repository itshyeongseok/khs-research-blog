Data-driven modeling of a flapping bat robot with a single flexible wing surface
Tier 1 — 최상위
Jonathan Hoff, Seth Hutchinson
Robotics: Science and Systems (RSS) 2020, Corvallis, Oregon, USA, July 12-16, 2020
논문 요약
본 논문은 박쥐 로봇 B2를 위한 하이브리드 제일원리 + 데이터 기반 동역학 모델을 개발하는데, 이 로봇은 별개의 날개/꼬리 표면이 아니라 날개와 꼬리에 걸쳐 펼쳐진 단일 유연 실리콘 막을 독특하게 가진다. 기존의 물리 기반 2D 종방향 모델에서 출발하여, 저자들은 네 가지 구조적 항 — 날개-꼬리 결합 함수, 몸체 공기역학 평판, 비선형 꼬리각 사상, 스트립 이론 공기역학 — 을 추가한 뒤, 다수의 꼬리 구동기 구성에 걸친 장기(1초) 비행 경로 예측을 최대화하도록 로드셀 및 자유 비행 데이터에 대해 모델 매개변수를 최적화한다. 단 9회의 로드셀 시험과 소수의 1초 비행만을 사용하여, 최적화된 모델은 원본 대비 피치와 고도 예측을 극적으로 개선하며, 저자들은 이를 자유 비행 데이터를 사용한 박쥐 형태 로봇의 장기 비행 예측에 대한 최초의 검증으로 보고한다. 프로젝트 맥락에서 이 논문은 (RL 제어가 아닌) 데이터 기반 *모델링* 대응물로서 날갯짓 축에 위치한다: 심각하게 과소구동된 날갯짓 비행체의 정확하고 물리적으로 의미 있는 동역학 모델은 모델 기반 학습, 궤적 최적화, 상태 추정의 전제 조건 — 학습/RL 비행 제어기가 궁극적으로 의존하는 기반 — 이다.

Introduction — 연구 동기 (클릭하여 자세히)
정확한 모델은 로봇공학에서 궤적 계획, 장기 예측, 제어, 상태 추정에 결정적이며, 특히 비정상 공기역학과 날개 유연성으로 고도로 과소구동된 날갯짓 시스템에 더욱 그러하다. 다수의 연구가 FWMAV(곤충 규모 비행체, 오니솝터, 로봇 벌새)를 모델링하지만, 능동 및 수동 관절을 가지며 날개와 꼬리에 걸쳐 펼쳐진 단일 유연 막을 가진 박쥐 형태 로봇을 다루는 연구는 극소수이다. 저자들은 박쥐 로봇 B2를 대상으로, 제일원리 추가와 데이터 기반 매개변수 최적화를 통해 기존 모델을 개선하여, 소수의 로드셀 시험과 1초 비행만으로 우수한 장기 예측을 달성한다.
Related Works — 관련 연구 (클릭하여 자세히)
본 논문은 날갯짓 비행의 시스템 식별을 조사한다: 선형 최소제곱 또는 경사 기반 비선형 최적화로 가속도 데이터를 사용하는 시간 영역 방법, 1-스텝 대 장기(순방향 적분) 예측, 그리고 특정 작동점 근처에서 효과적인 선형 모델 계열. 순수 데이터 기반 접근은 미탐색 상태에 대해 새 데이터를 필요로 하는 반면, 데이터로 조정된 매개변수를 가진 물리 기반 모델은 적응성이 있고 더 적은 실험을 요구하며 설계 최적화를 가능하게 한다 — 본 논문이 채택하는 전략이다.
Method — 방법론 (클릭하여 자세히)
B2의 기존 2D 종방향 오일러-라그랑주 모델(피치, x/z 위치, 날갯짓, 회내/회외, 등배 꼬리각의 형상과 준정상 받음각 공기역학)에서 출발하여, 저자들은 네 가지 추가를 도입한다. 날개-꼬리 결합 함수는 단일 막 물리를 포착하기 위해 기준 회내각을 스케일된 꼬리각만큼 증강한다. 몸체 공기역학 평판은 높은 받음각에서 몸체에 평평하게 고정된 막으로부터의 항력을 추가한다. 비선형 꼬리각 사상은 음이 아닌 꼬리각에 대해 양의 오프셋을 추가하여 중립 위치 근처의 급격한 민감도를 해결한다. 마지막으로 스트립 이론(3개 스트립)이 공기역학적 힘을 계산한다. 모든 새 매개변수는 데이터 기반 최적화에 맡겨진다.
Experiment — 실험 (클릭하여 자세히)
두 종류의 데이터가 수집된다: 6축 힘 센서에서 1000 Hz로 9개 날갯짓 주파수(2.5-7.8 Hz, 무풍)에 걸친 로드셀 힘 데이터, 그리고 8-카메라 Vicon 시스템이 100 Hz로 기록한 12개 꼬리 구성(q_DV -16~20도)의 43회 자유 비행 실험에 더해 탑재 IMU. IMU 방위(정확, 낮은 드리프트)와 Vicon 위치는 상호상관 시간 동기화 후 융합된다. 매개변수는 두 단계로 추정된다: 로드셀 데이터가 추력에 맞도록 회내 진폭을 조정한 뒤, fmincon(내부점법)이 여덟 개의 결정 변수를 최적화하여 비행 데이터에 대한 다단계 순방향 시뮬레이션 예측 오차를 최소화한다.
Discussion — 논의 (클릭하여 자세히)
최적화된 모델은 q_DV < 0, q_DV >= 0, 그리고 동적 꼬리 변경 비행에 걸쳐 원본 대비 장기 피치 및 고도 예측을 극적으로 개선한다. 날개-꼬리 결합 항은 필수적이다: 이것이 없으면 모델은 큰 양의 꼬리각에서 꼬리의 영향을 과대평가하고 고도를 너무 많이 얻는데, 위로 기운 꼬리가 날개를 양력을 줄이는 음의 기울기로 편향시키기 때문이다. 매개변수 변화(감소된 날개 코드와 꼬리 스팬, 따라서 감소된 면적/힘)는 물리적 설명을 가진다. 훈련 비행 수에 대한 절제 연구는 단 한 번의 1초 비행조차 극적인 개선을 주며, ~3회 시도 이후 수익 체감을 보인다.
Conclusion — 결론 (클릭하여 자세히)
저자들은 기존 물리 기반 구조를 여러 추가 항으로 확장하고 로드셀 및 자유 비행 실험으로 매개변수를 최적화하여 박쥐 로봇의 정확한 모델을 구축하는 데이터 기반 방법론을 개발했다. 장기 예측 결과는 다양한 꼬리 구성에 걸친 모델의 정확도를 입증한다. 한계는 물리적 수정이 재훈련을 위해 새 데이터를 요구할 수 있다는 것이지만, 물리 정보 기반 구조와 추가 항 덕분에 우수한 장기 예측에는 소수의 비행 실험 — 단 한 번의 1초 비행조차 — 만이 필요하다.
