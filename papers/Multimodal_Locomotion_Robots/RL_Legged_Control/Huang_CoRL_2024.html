DiffuseLoco: Real-Time Legged Locomotion Control with Diffusion from Offline Datasets
Tier 2 — 일반
Xiaoyu Huang, Yufeng Chi, Ruofeng Wang, Zhongyu Li, Xue Bin Peng, Sophia Shao, Borivoje Nikolic, Koushil Sreenath
Conference on Robot Learning (CoRL) 2024 (arXiv:2404.19264v1, Apr 2024) · codebase and checkpoints to be open-sourced upon acceptance
논문 요약
이 논문은 오프라인 데이터셋만으로 다중 스킬 확산(diffusion) 기반 보행 정책을 학습하여, 실제 다리형 로봇에서 다양한 민첩한 스킬의 실시간 제어를 가능하게 하는 프레임워크 DiffuseLoco를 제시한다. 단일 정책이 네 가지 사족 스킬(트로팅, 페이싱, 호핑, 바운싱)과 민첩한 이족 보행 스킬을 학습하고, 하드웨어로 제로샷(zero-shot) 전이되며, 데이터에 명시적으로 존재하지 않는 스킬 간에도 자유롭게 전환한다. 핵심 구조는 지연된 고유수용성(proprioceptive) 상태 및 목표 이력에 조건화되어 관절 수준 PD 목표 시퀀스를 노이즈 제거하는 트랜스포머 기반 잡음 제거 확산 확률 모델(Denoising Diffusion Probabilistic Model, DDPM)이다. 두 가지 설계 선택이 실시간 제어를 가능하게 한다: 빠른 재계획을 위한 후퇴 지평 제어(Receding Horizon Control, RHC)와, 현재 상태가 도착하기 전에 추론을 시작하여 대형 모델 지연을 숨기는 지연 입력 방식이다. 실세계 벤치마크에서 DiffuseLoco는 넘어지지 않고 모든 속도 추적 시험을 완료한 유일한 방법으로, 안정성과 추적 오차 모두에서 다중 스킬 RL(AMP)과 비확산 행동 복제 베이스라인을 능가한다. 다중모드 로봇의 RL 보행 제어 서브시스템에 대해, 이 연구는 여러 단일 스킬 RL 전문가로부터의 오프라인 데이터가 온라인 롤아웃이나 보상 엔지니어링 없이 다중 모드 보행 거동을 포착하는 하나의 표현력 있는 정책으로 증류될 수 있음을 보여준다.

Introduction — 연구 동기 (클릭하여 자세히)
대규모 오프라인 데이터셋으로부터의 학습은 비전, 언어, 로봇 조작에서 돌파구를 이끌었지만, 다리형 보행, 특히 하나의 정책에 여러 스킬을 학습하기 위한 학습 확장은 온라인 롤아웃의 비용 때문에 온라인 RL에는 어렵다. 대부분의 기존 보행 제어기는 특화된 단일 스킬 모델이며, 안정적인 이족 보행과 사족 호핑 같은 다양한 민첩 스킬을 하나의 정책에 결합하는 것은 시연된 적이 없었다. DiffuseLoco는 두 가지 과제를 동시에 해결하는 것을 제안한다: 다양한 데이터 소스로부터의 오프라인 학습과 다양한 스킬 집합의 학습이다. 수동 스킬 레이블링 없이 다양한 오프라인 데이터셋의 다중 모드성을 포착하기 위해 표현력 있는 확산 모델을 사용한 뒤, 학습된 제어기를 실제 다리형 로봇에서 강건한 실시간 제어를 위해 배치한다.
Related Works — 관련 연구 (클릭하여 자세히)
본 논문은 세 가지 흐름을 검토한다: 보행에서의 다중 스킬 RL, 보행 제어를 위한 오프라인 학습, 그리고 로봇공학에서의 확산 모델이다. 다중 스킬 RL은 고수준 계획이나 증류로 조정되는 별도 정책을 학습하거나, 매개화된 동작과 동작 모방(보상 정형 또는 적대적 모방)을 통해 직접 학습하지만, 안정적 이족 보행과 사족 호핑 같은 다양한 민첩 스킬을 하나의 정책에 결합하는 것은 보여지지 않았다. 오프라인 학습은 더 나은 확장성과 데이터 재사용을 제공하지만, 기존 보행 연구는 지나치게 단순화된 시뮬레이션 과제나 단일 스킬에 국한되었다. 확산 모델은 조작, 계획, 목표 생성에 적용되었지만, 다리형 로봇을 위한 고빈도 저수준 제어는 대체로 미탐구로 남아 있었다.
Method — 방법론 (클릭하여 자세히)
DiffuseLoco는 세 단계 프레임워크이다: 데이터 소스, 학습, 배치. (임의의 방법으로 얻은) 소스 단일 스킬 정책을 롤아웃하여 소스 불가지론적(source-agnostic) 고유수용성 상태, 관절 수준 PD 목표 행동, 목표 튜플을 오프라인 데이터셋으로 수집한다. 그다음 트랜스포머 기반 DDPM을 학습하여, 단순화된 MSE 잡음 예측 목적함수를 사용해 지연된 상태 및 목표 이력에 조건화된 행동 시퀀스를 노이즈 제거한다. 배치 시 정책은 제로샷으로 전이되어 후퇴 지평 제어 방식으로 실행된다: 미래 행동 시퀀스를 노이즈 제거하지만 즉각적인 행동만 실행하고, 새로운 상태로 빠르게 재계획한다. 대형 모델 지연에도 30-50 Hz 제어 요구를 충족하기 위해, 지연 입력은 현재 상태가 도착하기 전에 추론을 시작하게 하고, (DDIM 대신 유지된) DDPM은 안정적 실시간 제어를 위해 행동 정확도를 보존한다.
Experiment — 실험 (클릭하여 자세히)
실세계 평가는 두 질문에 답한다: DiffuseLoco가 다양한 시연 소스로부터 학습될 수 있는가, 그리고 다양한 스킬 집합을 통합할 수 있는가. 단일 사족 정책이 AMP, CPG 곡선, 대칭 증강으로 학습된 소스로부터 다섯 스킬(트로팅, 페이싱, 호핑, 바운싱, 민첩한 이족 보행)을 학습하고, 하드웨어로 제로샷 전이되어, 5회 연속 실행에서 실패 없이 스킬 간에 자유롭게 전환한다. 다섯 가지 속도/선회 명령 하에 4미터 보행을 측정하는 정량 벤치마크는 다중 스킬 RL(AMP, 이력 포함 AMP)과 비확산 BC 베이스라인(자기회귀 트랜스포머 TF, RHC 포함 TF) 대비 안정성(무낙상 시험)과 속도 추적 오차 E_v를 측정한다. DiffuseLoco는 넘어지지 않고 모든 시험을 완료한 유일한 방법으로, AMP를 무력화하는 모드 붕괴를 피하고, RHC 포함 TF보다 평균 38.97% 낮은 추적 오차를 달성한다. 절제 실험은 RHC, 도메인 무작위화, DDIM 대비 DDPM 샘플링, 트랜스포머 백본이 각각 기여함을 확인한다.
Discussion — 논의 (클릭하여 자세히)
DiffuseLoco는 다리형 보행 학습에 대한 확장 가능한 접근을 시연한다: 제어를 표현력 있는 확산 백본을 가진 오프라인 모방으로 취급함으로써, 다른 관측/행동 공간을 가진 다양한 RL 알고리즘의 시연을 흡수하고 동일 명령 하에서 여러 모드를 실행한다. 다른 다중 스킬 정책과 비교하여, 비확산 BC보다 더 부드러운 행동과 더 나은 sim-to-real 전이를 제공하고 AMP 같은 적대적 방법의 모드 붕괴를 피한다. 대규모 오프라인 데이터셋 패러다임은 시뮬레이션에 묶인 온라인 RL을 넘어 확장 가능한 실세계 데이터 수집과 학습으로 가는 실용적 경로를 제공한다. 저자들은 다른 로봇 형태로의 적응과 비전/언어 목표의 통합을 유망한 확장으로 가정하면서, 강건성이 여전히 데이터셋 다양성에 의존함을 인정한다.
Conclusion — 결론 (클릭하여 자세히)
DiffuseLoco는 다중 모드 오프라인 데이터셋으로부터 다양한 민첩 다리형 보행 스킬을 학습하고 실세계 로봇으로 실시간으로 강건하게 전이하는 확장 가능한 학습 프레임워크이다. 확산 모델을 활용하여 오프라인 데이터의 다중 모드성을 포착함으로써, 이족 보행과 사족 스킬을 하나의 정책 안에 결합한 최첨단 제어기를 학습하고 그들 사이를 자유롭게 전환한다. 처음으로, 저자들은 그러한 다양한 오프라인 학습 보행 정책을 실세계 응용으로 제로샷 전이하는 것이 실현 가능함을 보여주어, 데이터셋을 지속적으로 확장하고 다양한 데이터 소스로부터의 다양한 스킬을 학습 기반 제어를 위해 통합할 가능성을 연다.
