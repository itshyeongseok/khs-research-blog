Efficiently Learning Robust Torque-based Locomotion Through Reinforcement with Model-Based Supervision
Tier 1 — 최상위
Yashuai Yan, Tobias Egle, Christian Ott, Dongheui Lee
IEEE Robotics and Automation Letters (RAL), 2026 (arXiv:2510.16190v1, 22 Jan 2026). Tier 1 (RAL, JCR Q1). Evaluation is simulation-only (MuJoCo); no physical-robot experiments reported in this version.
논문 요약
본 논문은 모델 기반 이족 이동과 잔차 강화학습(residual reinforcement learning)을 융합하여 실세계 불확실성 하에서 견고하고 적응적인 토크 수준 보행을 달성하는 제어 프레임워크를 제안한다. 모델 기반 제어기 — 운동 발산 성분(Divergent Component of Motion)(DCM) 궤적 계획기와 역동역학 전신 제어기(WBC) — 가 신뢰성 있는 Base 정책 역할을 하고, 그 위에서 학습 가능한 잔차 정책이 모델링되지 않은 효과(관절 마찰, 구동기 비선형성, 백래시, 센서 잡음)를 보상한다. 핵심 아이디어는 학습 중에만 실제 동역학과 무작위화 파라미터에 특권적 접근을 갖고 이상적 보정 토크를 생성하는 모델 기반 Oracle 정책이다. 이 Oracle은 표준 PPO 목적과 결합된 새로운 지도 손실(weights omega_rl=1, omega_sup=10)을 통해 잔차 정책을 감독한다. 센서·로봇 동역학·지형에 대한 도메인 무작위화가 정책을 실세계 가변성에 노출한다. MuJoCo에서 세 이족 로봇 — Kangaroo(47 kg), Unitree H1-2(67 kg), Bruce(4.8 kg) — 으로 평가한 결과, 본 방법(BOR)은 로봇별 튜닝 없이 동일한 학습 설정으로 세 로봇 모두에서 Oracle에 근접한 DCM/발 추종과 함께 100% 성공을 달성하고, 비지도 잔차 RL보다 훨씬 빠르게 학습하며(단일 환경에서 약 1,500 에피소드 / 4.16시간 후 80% 초과 성공), 모델 기반 제어기가 10%로 떨어지는 최고 무작위화 수준까지 100% 성공을 유지한다.

Introduction — 연구 동기 (클릭하여 자세히)
이족 로봇의 토크 기반 이동 학습은 핵심 과제이다: 토크 제어는 반응성 있고 에너지 효율적이며 순응적인 거동을 가능하게 하지만 잘못 조절되면 불안정하며, 특히 접촉 불확실성, 센서 잡음, 구동 지연 하의 고자유도 부유 기저 시스템에서 그러하다. 전통적 모델 기반 제어는 이동을 상위 궤적 계획과 하위 전신 제어로 분해하지만 모델-플랜트 불일치와 잡음 섞인 추정 하에서 성능이 저하된다. 저자들은 학습 목적에 명시적 지도 손실을 추가하는 지도 강화학습 프레임워크의 동기를 제시하며, 모델 부정확성을 보상하는 잔차 RL 정책으로 모델 기반 제어기를 증강하고, 도메인 무작위화로 실세계 불확실성에 노출하며 특권 Oracle로 안내한다.
Related Works — 관련 연구 (클릭하여 자세히)
본 논문은 이족 이동에서 실세계 불확실성을 다루는 선행 전략을 검토하고 자신의 지도 잔차 RL을 이에 대비하여 위치시킨다. 반복 학습 제어(예: 보상적 ZMP 궤적 학습)는 반복 실행을 통해 참조를 정련하지만 일관되고 반복적인 동작 패턴을 필요로 한다. 도메인 무작위화를 동반한 심층 RL은 견고성을 주지만 데이터를 많이 요구하고 보상에 민감하다. 최근 하이브리드 방법은 로봇 모델 지식을 DRL에 통합하여(저수준 역동역학을 갖춘 작업 공간 정책, 각운동량 LIPM 영감 계층, 디딤 위치/타이밍 적응을 위한 RL) 보상 공학을 줄이고 샘플 효율을 개선하지만, 여전히 보상 신호에서만 학습하고 학습 시 제한적 안내만 제공한다 — 본 연구가 명시적 Oracle 감독으로 메우는 간극이다.
Method — 방법론 (클릭하여 자세히)
본 방법은 모델 기반 베이스 제어기와 지도 잔차 RL 정책을 통합한다. 모델 기반 제어기는 DCM/가상 반발점(Virtual Repellent Point) 궤적 생성기와, 불안정한 1차 DCM 동역학을 안정화하고 원하는 질량중심 힘을 명령하는 역동역학 전신 제어기로 구성된다. 도메인 무작위화는 센서 잡음, 동역학(질량, 관성, 모터 강도, 구동 지연), 환경(바닥 마찰, 불균일)에 걸친다. 세 정책이 정의된다: 부정확한 모델과 잡음 섞인 관측 하의 Base 정책, 학습 중에만 이상적 보정 토크를 생성하는 실제 동역학을 갖춘 특권 Oracle 정책, 그리고 학습 가능한 잔차 정책이다. 잔차 정책은 PPO와, Oracle의 보정 토크를 모방하도록 장려하는 지도 손실로 학습되며, 총 손실은 모방 대 탐색을 가중한다(omega_rl=1, omega_sup=10).
Experiment — 실험 (클릭하여 자세히)
프레임워크는 MuJoCo에서 세 이족 로봇 — Kangaroo(47 kg, 145 cm, 다리당 6 DoF, 평발), Unitree H1-2(67 kg, 178 cm, 다리당 6 DoF, 평발), Bruce(4.8 kg, 70 cm, 다리당 5 DoF, 선형 발) — 으로 평가되며, PIQP 전신 QP를 약 1 ms에 풀고 시뮬레이션 1000 Hz, 제어 200 Hz, RL은 10k 에피소드(각 10초)에 걸쳐 학습한다. 기준선은 모델 기반 제어기(MBC), 비지도 잔차 RL(omega_sup=0), 모방 학습(omega_rl=0)을 포함한다. 전체 방법(BOR)은 Oracle에 근접한 추종과 함께 세 로봇 모두에서 100% 성공을 달성하고, 비지도 잔차 RL보다 훨씬 빠르게 학습하며, MBC가 무너지는 모든 무작위화 수준에서 100% 성공을 유지한다. 절제 실험은 잔차 학습이 직접 토크 학습보다 빠르게 수렴함을 보이나, Base와 Oracle이 크게 갈릴 때는 직접 학습 대비 이득이 제한적이다.
Discussion — 논의 (클릭하여 자세히)
논의는 본 접근법을 두 패러다임을 잇는 것으로 규정한다: 데이터 비효율적인 토크 수준 RL과, 거의 정확하지 않은 시스템 모델에 의존하는 모델 기반 최적화이다. 시뮬레이션에서 실세계 불확실성을 명시적으로 모델링하며 Oracle이 안내하는 잔차 토크 정책을 학습하고, 강화와 감독 목적을 공동 최적화함으로써, 본 방법은 모방과 탐색을 균형 짓는다. Oracle 안내는 학습 효율을 크게 개선하지만, 저자들은 실제 정보를 아는 Oracle조차 모든 불리한 조건을 예측할 수 없으며 — 급격한 명령 변화나 불안정한 지면 접촉이 실패를 유발함 — Oracle에 대한 과의존이 Bruce와 H1에서 보이듯 매우 확률적인 조건에서 성능을 제한할 수 있음을 강조한다.
Conclusion — 결론 (클릭하여 자세히)
본 논문은 모델 기반 제어 고유의 sim-to-real 간극과 모델 프리 RL의 데이터 비효율을 모두 다루는 토크 기반 휴머노이드 이동을 위한 지도 강화학습 프레임워크로 결론짓는다. 학습 중 도메인 무작위화는 일반화를 넓히고, 모델 기반 Oracle 정책은 무작위화 동역학 하에서 차선이지만 유익한 감독을 제공하며, 이는 경사 갱신 중 정책을 Oracle과 명시적으로 정렬하는 추가 손실 항을 통해 통합된다. Kangaroo, Unitree H1-2, Bruce에 동일 프레임워크와 로봇별 수정 없이 평가했을 때 학습된 정책은 각자의 Oracle에 필적하는 성능을 달성하며, 절제 실험은 프레임워크가 베이스 위 잔차 학습과 직접 토크 명령 학습을 모두 지원하여 효율적이고 견고하며 전이 가능한 해를 제공함을 확인한다.
