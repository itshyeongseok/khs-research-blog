Sim-to-Real: Learning Agile Locomotion For Quadruped Robots
Tier 1 — 최상위
Jie Tan, Tingnan Zhang, Erwin Coumans, Atil Iscen, Yunfei Bai, Danijar Hafner, Steven Bohez, Vincent Vanhoucke
Robotics: Science and Systems (RSS) XIV, 2018. DOI:10.15607/RSS.2018.XIV.010. Preprint arXiv:1804.10332.
논문 요약
저자들은 민첩한 사족 보행 제어 policy를 전적으로 물리 시뮬레이터(PyBullet)에서 학습한 뒤, 추가 on-robot 학습 없이 실제 Minitaur direct-drive 로봇에 배포하는 완전한 시스템을 제시한다. 핵심 문제는 reality gap이다: 시뮬레이션에서 학습된 policy는 모델 불일치 때문에 보통 하드웨어에서 실패한다. 본 논문은 이 gap을 상보적 두 방향에서 공략한다. 첫째, system identification, 정확한 URDF, 이상적 DC 모터 dynamics에서 유도하고 구간 선형(piece-wise-linear) 토크-전류 포화 곡선을 가진 물리 기반 actuator model, 그리고 보간을 동반한 시각 표기(time-stamped) 관측 history를 통한 명시적 latency 모델링으로 시뮬레이션 충실도를 높인다. 둘째, 물리/dynamics 파라미터를 randomize하고, 무작위 외란 force를 주입하며, 시뮬레이션 인공물에 과적합하지 않도록 의도적으로 compact(저차원) 관측 공간을 써서 robust한 제어기를 학습한다. 제어는 Proximal Policy Optimization으로 푸는 POMDP로 정식화되며, action 공간은 개루프(open-loop) reference + 학습된 feedback의 하이브리드로 완전 지정에서 처음부터 학습까지의 제어 가능성(controllability) 스펙트럼을 사용자에게 제공한다. 학습된 trotting과 galloping gait는 하드웨어로 직접 전이되며 비슷한 속도에서 전문가 수작업 gait보다 에너지 효율이 높다. Ko의 MARCO 연구에 대해 본 논문은 dynamics reality gap을 메우는 actuator-model + domain-randomization + compact-observation 레시피의 정전적 인용 문헌이다.

Introduction — 연구 동기 (클릭하여 자세히)
Introduction은 민첩한 사족 보행을 오랜 난제로 규정한다. 로봇이 under-actuated이고 균형 집약적인 고도로 동적인 운동을 수행해야 해 전통적으로 전문가 경험과 지루한 수작업 튜닝을 요구하기 때문이다. deep RL은 자동화를 제공하지만 거의 항상 시뮬레이션에 국한되며, 거기서 학습된 제어기는 reality gap—보행의 빈번한 contact 변화로 증폭되는, 시뮬레이션과 현실 사이의 모델 불일치—때문에 하드웨어로 잘 전이되지 않는다. 보행을 위한 on-robot 학습은 느리고 비싸며 낙상 시 로봇 손상 위험이 있으므로, 저자들은 시뮬레이션 학습이 더 매력적이라 주장하고, 시뮬레이터 개선과 robust 제어기 학습 둘 다로 gap을 좁히며 시뮬레이션에서 학습해 실제 로봇에 배포하는 완전한 시스템을 제시한다.
Related Works — 관련 연구 (클릭하여 자세히)
관련 연구는 두 갈래를 다룬다: legged 보행 제어, 그리고 reality gap 극복. 제어에서는 고전적 제어기 최적화(실제 로봇에 데이터 효율적이나 고차원 제어로 잘 확장되지 않는 black-box·Bayesian 최적화)와 사람 전문성 필요를 줄이는 deep RL을 대비한다. 저자들은 안정성과 손쉬운 병렬화 때문에 PPO를 택한다. reality gap에 대해서는 system identification, online/암묵적 파라미터 식별, 노이즈 주입·외란·다중 시뮬레이터를 통한 robust성, domain randomization과 dynamics randomization, 그리고 시뮬레이션+실제 데이터 모두로부터의 학습(fine-tuning, action 공간 변환, 시각 domain adaptation)을 개관하며, 인지 중심 전이 기법은 gap이 인지가 아니라 dynamics에 있는 보행에 적용하기 어렵다고 지적한다.
Method — 방법론 (클릭하여 자세히)
로봇 플랫폼은 Minitaur로, 8개의 direct-drive actuator(다리당 2개)를 가진 사족이며 Nvidia Jetson TX2가 신경망 추론을 수행해 UART로 STM32 마이크로컨트롤러와 통신하며 가변 150-200 Hz로 제어된다. 시뮬레이션은 PyBullet으로 구축된다. 제어는 PPO로 푸는 POMDP로 정식화되고, policy는 기대 return을 최대화한다. 관측 공간은 의도적으로 compact하며(IMU roll, pitch와 그 축들에 대한 각속도, 그리고 모터 각도), action 공간은 leg-space의 swing/extension 자세를 두 모터 각도로 매핑해 자기 충돌하는 무효 구성을 가지치기한다. 보상은 전진 거리를 보상하고 에너지를 벌점한다. policy는 개루프 reference 성분 + 학습된 feedback 성분으로 분해되어 제어 가능성 스펙트럼을 주며, feedback policy는 hidden layer 2개의 완전연결 network다.
Experiment — 실험 (클릭하여 자세히)
reality gap은 시뮬레이션 충실도 향상과 robust 제어기 학습으로 좁혀진다. 시뮬레이션 충실도는 정확한 URDF(측정 질량, 치수, 추정 관성, 측정 모터 마찰), 이상적 DC 모터 dynamics에서 유도하고 구간 선형 토크-전류 포화 곡선을 가진 충실한 actuator model, 그리고 선형 보간을 동반한 시각 표기 관측 history를 쓴 명시적 latency 모델링으로 향상된다. robust성은 신중히 선택한 범위에서 10개 물리 파라미터를 randomize하고, base에 무작위 외란 force를 더하며, compact 관측 공간을 쓰는 데서 온다. 평가는 reality gap을 시뮬레이션과 실제의 기대 return 차로 정의하며, 100개 학습 제어기에 걸친 ablation(상위 3개 배포, 각 3회 실행)은 시뮬레이션 개선이 필수적이고, randomization이 최적성을 robust성과 맞바꾸며, compact 관측 공간이 최고의 실세계 전이를 줌을 보인다. 학습된 trotting과 galloping은 직접 전이되며 전문가 수작업 gait보다 전력을 23-35% 적게 쓴다.
Discussion — 논의 (클릭하여 자세히)
논의(평가와 엮임)는 기법들이 왜 작동하는지 해석하고 핵심 trade-off를 드러낸다. 시뮬레이션 충실도 향상이 지배적 지렛대다: 모델 불일치가 너무 크면 무작위 외란으로 robust화된 제어기조차 gap을 건너지 못하며, 정확한 actuator model과 latency 시뮬레이션 둘 다 각각 필요하다. randomization은 robust성을 높이고 gap을 좁히지만 공짜가 아니다: 평균 return(보수적·차선 거동)과 그 분산(robust성)을 모두 낮추므로 필요할 때만 적용해야 한다. 관측 공간 선택은 미묘하나 결정적 요인인데, 고차원 관측 공간은 시뮬레이션 성능은 높이지만 실세계 관측 분포를 희소한 학습 분포에서 벗어나게 만들어 reality gap을 넓히므로, 전이에는 compact 관측 공간이 선호된다.
Conclusion — 결론 (클릭하여 자세히)
결론은 deep RL이 민첩한 사족 보행을 자동으로 학습할 수 있고, 정확한 물리 모델과 robust 제어기로 시뮬레이션에서 학습한 본 완전 시스템이 시뮬레이션 학습 제어기를 실제 로봇에 성공적으로 배포함을 밝힌다. 이 시스템으로 저자들은 Minitaur를 위한 두 민첩한 gait, trotting과 galloping을 개발했다. 초점은 단순 보상과 단순 환경(평지 전진 주행 속도 최대화) 하의 전이 가능한 보행이었음을 밝히고, 두 향후 방향을 제시한다: 주행 속도와 방향을 동적으로 바꾸는 policy 학습, 그리고 vision을 감각 입력으로 통합한 복잡 지형 확장.
