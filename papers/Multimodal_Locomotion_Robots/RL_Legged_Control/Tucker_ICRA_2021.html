Preference-Based Learning for User-Guided HZD Gait Generation on Bipedal Walking Robots
Tier 1 — 최상위
Maegan Tucker, Noel Csomay-Shanklin, Wen-Loong Ma, Aaron D. Ames
2021 IEEE International Conference on Robotics and Automation (ICRA) (arXiv:2102.04450v2, Mar 2021) · open-source code at github.com/maegant/ICRA2021-LearningHZD
논문 요약
이 논문은 제어 이론과 기계학습을 융합하여 수동 파라미터 튜닝 없이 안정적이고 강건한 이족 보행을 얻는 프레임워크 LINECOSPAR-NLP를 제시한다. 보행은 하이브리드 영동역학(hybrid zero dynamics, HZD) 궤적 최적화(FROST/IPOPT를 통한 직접 콜로케이션)로 생성되며, 그러한 보행을 형성하는 다섯 가지 '필수 제약(essential constraints)'(전진 속도, 발 들림 위상, 최소 발 들림, 충격 속도, 보폭)은 인간의 쌍별(pairwise) 선호로부터 잠재 효용 함수에 대한 베이즈 사후분포를 추론하는 고차원 선호 기반 학습(preference-based learning) 알고리즘으로 튜닝된다. 결정적으로 이는 보상 함수의 수작업 설계를 피하고 시뮬레이션을 전혀 사용하지 않는다. 평면 이족 로봇 AMBER-3M에서의 두 하드웨어 실험—첫째는 강체 점접촉 발(point-feet), 둘째는 의도적으로 모델링되지 않은 컴플라이언트 스프링 발(spring-feet)—은 50회 미만의 반복으로 안정적이고 강건하며 효율적이고 자연스러운 보행을 달성한다. 다중모드 로봇의 RL 구동 지상 보행 서브시스템에 대해, 이 연구는 보상 엔지니어링에 대한 보상 없는(reward-free) 샘플 효율적 대안으로, 운영자 선호가 하드웨어에서 실제 이족 보행을 직접 튜닝할 수 있음을 보여준다.

Introduction — 연구 동기 (클릭하여 자세히)
하드웨어에서의 동적 이족 보행은 제어, 공학, 기계학습에 걸친 벤치마크 문제로 남아 있다. 고전 제어 접근(ZMP, LIP, SLIP, 중심체 동역학)과 HZD 프레임워크는 안정성 보장을 제공하지만 모델링되지 않은 동역학에 대한 강건성을 요구하며, 이는 역사적으로 고된 파라미터 튜닝을 통해 얻어졌다. 기계학습 접근(RL, 모방 학습)은 수동 튜닝을 피하지만 정성껏 설계된 보상 함수와 현실과 어긋나는 물리 엔진에 의존한다. 본 논문은 제어 이론의 형식적 안정성을 복잡한 파라미터 조합을 보행 거동으로 매핑하는 학습과 결합할 것을 제안하며, 인간의 쌍별 선호로부터 HZD 보행을 튜닝하기 위해 선호 기반 학습을 사용한다.
Related Works — 관련 연구 (클릭하여 자세히)
본 연구는 외골격 보행 최적화를 위한 저자들의 기존 선호 기반 학습과 걷기, 달리기, 사족 보행을 위한 더 넓은 HZD 문헌에 기반한다. 모델 기반 제어(안정성 보장을 제공하지만 튜닝이 필요)와 강화학습 및 모방 학습(수동 튜닝을 피하지만 보상 함수와 물리 엔진이 필요)을 대조한다. Self-Sparring / 다중 결투(multi-dueling) 밴딧과 가우시안 과정 선호 학습이 LINECOSPAR-NLP가 확장하는 알고리즘적 기반을 제공한다.
Method — 방법론 (클릭하여 자세히)
제어 방식은 두 기둥에 기반한다: 전차수 동역학을 영동역학 다양체로 축소하여 보행 안정성을 인증하는 HZD와, 베지에(Bezier) 계수로 매개화된 안정적 주기 보행을 생성하는 직접 콜로케이션 궤적 최적화(FROST/IPOPT)이다. 보행은 연속 동역학과 이산 충격 리셋을 갖는 하이브리드 제어 시스템으로 인코딩되며, 하이브리드 불변(HZD) 조건이 성립해야 한다. 보행을 형성하는 다섯 가지 '필수 제약'은 허용 대역(tolerance band)으로 재정식화되어 LINECOSPAR-NLP로 튜닝되며, 이는 쌍별 선호로부터 잠재 효용 함수에 대한 베이즈 사후분포를 모델링하고 무작위 선형 부분공간을 따라 Self-Sparring 톰슨 샘플링으로 새로운 보행을 선택한다.
Experiment — 실험 (클릭하여 자세히)
LINECOSPAR-NLP는 평면 이족 로봇 AMBER-3M에서 다섯 가지 필수 제약을 두 하드웨어 구성으로 튜닝하도록 배치된다: 강체 점접촉 발(AMBER3M-PF)과 컴플라이언트 스프링 발(AMBER3M-SF)이며, 후자는 컴플라이언스를 모델과 제어기에서 의도적으로 제외했다. 제어는 오프보드에서 약 1 kHz로 실행되고, 보행은 5차 베지에 다항식과 기계적 운송 비용(mechanical cost of transport, MCOT)을 최적화 비용으로 사용한다. 반복마다 두 보행을 비교하며(n=2), 선호는 걷기 능력, 강건성, 소음, 시각적 매력으로 순위가 매겨진다. 두 실험 모두 50회 반복 이내에 안정적이고 강건하며 효율적인 보행으로 수렴한다.
Discussion — 논의 (클릭하여 자세히)
실험은 선호 기반 학습이 보상 엔지니어링이나 시뮬레이션 없이, 상당한 모델 불확실성 하에서도 하드웨어에서 동적으로 안정적인 걷기를 체계적으로 실현할 수 있음을 시연한다. 점접촉 발 사례는 효율적이고 강건한 보행으로의 수렴을 보이는 한편, 스프링 발 사례는 프레임워크가 역사적으로 정교한 모델을 요구했던 모델링되지 않은 컴플라이언스를 보상할 수 있음을 보인다. 이 연구는 또한 운영자가 직접 측정하지 않고도 식별할 수 있는 잠재적 효율 향상(더 낮은 MCOT)을 드러내어, 다요인 최적화 신호로서 인간 판단의 강점을 예시한다.
Conclusion — 결론 (클릭하여 자세히)
본 논문은 HZD 보행 생성을 위해 특별히 구축된 고차원 선호 기반 학습 프레임워크 LINECOSPAR-NLP를 제시하고 실험적으로 검증하며, 이는 파라미터 튜닝의 부담 없이 HZD의 이론적 이점을 제공한다. 제공하기 쉬운 쌍별 선호만을 사용하여, 모델링되지 않은 컴플라이언트 다리를 포함하여 50회 하드웨어 반복 이내에 시뮬레이션 없이 안정적이고 강건하며 효율적이고 자연스러운 걷기를 실현한다. 저자들은 프레임워크를 사족 로봇과 3D 이족 로봇으로, 그리고 더 큰 샘플 효율성을 위한 더 풍부한 피드백(순서형 레이블)으로 확장하는 것을 지적하며, 결과를 학습과 제어의 경계에서의 진전으로 규정한다.
